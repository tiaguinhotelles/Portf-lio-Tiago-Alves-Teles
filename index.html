Sobre mim

Estudante de Engenharia de Qualidade de Software, experiência em processos de qualidade no desenvolvimento de software, linguagem de programação Java e Javascript, testes automatizados em projeto ágil, utilizando metodologia Scrum. Conhecimentos em Selenium, Cucumber com padrão page objects. Os testes de API são desenvolvidos em Restassured, ou executados via Postman e conhecimentos em testes UI desenvolvidos em um projeto Maven. No projeto, utilizo JIRA como ferramenta de ALM para gestão das atividades, git, github e bitbucket para o versionamento do código de testes automatizados. Minha missão era identificar e definir os testes exigidos, monitorar o processo de teste em detalhes e os resultados em cada ciclo de teste e avaliar a qualidade geral.
